## 快速开始 · Quick Start

1. **装服务端** — 在电脑上下载并安装上面的服务端程序，运行它。
   Install and run the server app on your computer.
2. **装手机端** — 在 iPhone 上安装 Sidekey App。
   Install the Sidekey app on your iPhone.
3. **配对** — 确保手机和电脑在同一局域网，按 App 内提示扫码/选择电脑完成配对。
   Make sure both devices are on the same network, then pair by following the in-app prompts.
4. 开始用 iPhone 当快捷键盘 / 麦克风。
   Start using your iPhone as a shortcut deck and microphone.

> macOS 首次运行需在「系统设置 → 隐私与安全性 → 辅助功能」中授权 Sidekey，才能注入按键/鼠标。
> On macOS, grant Sidekey access under System Settings → Privacy & Security → Accessibility on first run.

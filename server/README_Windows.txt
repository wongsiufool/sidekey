========================================
 Sidekey 电脑端 · Windows 使用说明
========================================

【方式一】打包成单个 exe，双击运行（推荐，运行时不需要装 Python）
--------------------------------------------------------
1. 先装一次 Python：到 python.org 下载 3.10+，安装时务必勾选
   "Add python.exe to PATH"。（这是给“打包”用的，打完包就不再需要。）
2. 双击  build_exe.bat ，等它跑完（1-3 分钟，需要联网）。
3. 完成后这个文件夹里会出现  SidekeyServer.exe 。
4. 以后双击  SidekeyServer.exe  就能启动。
   这个 exe 还能拷到别的 Windows 电脑直接用（那台不用装 Python）。

【方式二】不打包，直接双击运行（更简单，但需要装 Python，且会留一个 .venv 文件夹）
--------------------------------------------------------
双击  run_server.bat  即可。第一次会自动装环境，之后双击就启动。

========================================
 第一次启动会遇到的两件事（很重要）
========================================
1. 防火墙弹窗：会弹出 “Windows Defender 防火墙”，
   一定要勾选【专用网络】并点【允许访问】，否则手机连不上。
   （错过了：Windows 安全中心 → 防火墙 → 允许应用 → 找到它 → 勾上专用网络）

2. 蓝色拦截窗 “Windows 已保护你的电脑”（SmartScreen）：
   这是因为 exe 没有数字签名，属正常。点【更多信息】→【仍要运行】。

========================================
 怎么连手机
========================================
- 手机和这台 Windows 连同一个 Wi-Fi。
- 启动后窗口里会显示二维码 + 配对码 + 本机地址（192.168.x.x : 8765）。
- 手机打开 Sidekey App → 点左上电脑名 → “扫码添加电脑”，扫窗口里的二维码即可。
  （二维码花了的话：扫自动弹出的图片，或在 App 用“手动添加”粘贴配对码。）

启动那个黑色窗口要一直开着（它就是服务端）；关掉窗口 = 停止服务。
